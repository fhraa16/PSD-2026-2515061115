# Sistem Pencarian Letak Nomor Kursi Bioskop

Jadi sistem ini dibuat untuk mencari letak nomor kursi bioskop. Sistem ini menerapkan algoritma binary search buat nyari posisi nomor kursi tertentu yang sudah diurutkan di dalam listnya.
